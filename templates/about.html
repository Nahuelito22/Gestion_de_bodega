Acerca de este Proyecto

{# Separador #}

Esta aplicación web está diseñada para ofrecer una **gestión integral y una trazabilidad completa** del proceso de producción vinícola. Desde la recepción de la uva hasta el embotellado final, nuestra plataforma permite un monitoreo detallado de cada etapa, asegurando la calidad y el control de cada lote de vino.

Ha sido desarrollada como un proyecto práctico, enfocándose en la implementación de una arquitectura **CRUD (Crear, Leer, Actualizar, Eliminar)** utilizando el framework Flask en Python y SQLAlchemy para la gestión de la base de datos.

Nuestro objetivo es proporcionar una herramienta robusta y fácil de usar para bodegas, que facilite la toma de decisiones basada en datos y optimice la cadena de valor del vino.
